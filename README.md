jogodots
=======================================
Dots board game to play in server/client mode developed in 2003.

TOC
---
- [1 Server](#1-server)
- [2 Client](#2-client)

1 Server
--------

To execute the server run the following command:
```
java -classpath .\bin\ jogodots.FrameDotsServer
```

Click on button "Connect" to accept connections.
